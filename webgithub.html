GitHub Course

Learn GitHub from Basic to Advanced Concepts

1. Introduction to GitHub

  • What is GitHub?
    GitHub is a platform for version control and collaborative software development using Git.
  • Git vs. GitHub
    Git is a version control system; GitHub is a cloud-based service hosting Git repositories.
  • Creating a GitHub Account
    Sign up on GitHub to create a profile and access repository management tools.

2. Setting Up GitHub

  • Configuring Git for GitHub
    Set up Git with your name and email to link commits to your GitHub account.
  • Setting Up SSH Keys
    Generate SSH keys to securely authenticate GitHub operations without passwords.
  • Installing GitHub Desktop
    Use GitHub Desktop for a GUI to manage repositories and commits easily.

3. Creating and Managing Repositories

  • Creating a New Repository
    Initialize a new repository on GitHub to store and track project files.
  • Cloning Repositories
    Clone a repository to your local machine to work on its files offline.
  • Repository Settings and Visibility
    Adjust repository settings to make it public, private, or restricted to collaborators.

4. Basic GitHub Workflow

  • Pushing and Pulling Changes
    Push local commits to GitHub and pull updates from remote repositories.
  • Committing and Pushing via GitHub Desktop
    Use GitHub Desktop to commit changes and push them to your repository.
  • Using .gitignore Files
    Create a .gitignore file to exclude unnecessary files from version control.

5. Branching and Merging on GitHub

  • Creating and Managing Branches
    Create branches to work on features or fixes without affecting the main branch.
  • Merging via Pull Requests
    Use pull requests to review and merge branch changes into the main repository.
  • Resolving Merge Conflicts
    Resolve conflicts manually when merging branches with conflicting changes.

6. Collaboration on GitHub

  • Forking Repositories
    Fork a repository to create a personal copy for experimenting or contributing.
  • Contributing to Open Source
    Contribute to open-source projects by submitting pull requests with your changes.
  • Managing Collaborators
    Invite collaborators to a repository and assign roles for teamwork.

7. Pull Requests and Code Reviews

  • Creating Effective Pull Requests
    Write clear pull requests with detailed descriptions to facilitate code reviews.
  • Reviewing Code on GitHub
    Review pull requests, provide feedback, and approve changes on GitHub.
  • Using Pull Request Templates
    Use templates to standardize pull request details for consistency.

8. Issues and Project Management

  • Creating and Managing Issues
    Create issues to track bugs, tasks, or enhancements in a repository.
  • Using GitHub Projects for Task Tracking
    Organize tasks with GitHub Projects using boards, lists, or tables.
  • Labels and Milestones
    Use labels and milestones to categorize and prioritize issues effectively.

9. GitHub Actions

  • Introduction to GitHub Actions
    GitHub Actions automates workflows for testing, building, and deploying code.
  • Creating Workflows for CI/CD
    Set up CI/CD pipelines with GitHub Actions to automate testing and deployment.
  • Automating Tasks with Actions
    Use pre-built or custom actions to automate repetitive development tasks.

10. GitHub Pages

  • Hosting Static Websites
    Host static websites directly from a GitHub repository using GitHub Pages.
  • Configuring GitHub Pages
    Enable GitHub Pages and select a branch for hosting your site.
  • Custom Domains with GitHub Pages
    Link a custom domain to your GitHub Pages site for a personalized URL.

11. Advanced GitHub Features

  • Using GitHub CLI
    Use GitHub CLI to manage repositories, issues, and PRs from the terminal.
  • Managing GitHub Organizations
    Create and manage organizations to coordinate team projects and permissions.
  • GitHub Apps and Integrations
    Build or use GitHub Apps to integrate tools and enhance workflow automation.

12. Security on GitHub

  • Enabling Two-Factor Authentication
    Enable 2FA to secure your GitHub account with an extra authentication layer.
  • Dependabot for Dependency Updates
    Use Dependabot to automate dependency updates and fix vulnerabilities.
  • Code Scanning and Secret Scanning
    Enable code and secret scanning to detect vulnerabilities and exposed credentials.

13. GitHub Packages

  • Publishing Packages
    Publish reusable code packages to GitHub Packages for project sharing.
  • Using GitHub Packages with npm
    Integrate GitHub Packages with npm to manage JavaScript dependencies.
  • Managing Package Access
    Control access to packages by setting permissions for users or teams.

14. GitHub Enterprise and Teams

  • GitHub Enterprise Overview
    GitHub Enterprise offers advanced features for large-scale, secure development.
  • Managing Teams and Permissions
    Organize teams and set granular permissions for collaborative projects.
  • Enterprise Security Features
    Use SAML, LDAP, and audit logs for enhanced security in GitHub Enterprise.

15. GitHub Ecosystem and Community

  • Exploring GitHub Marketplace
    Discover tools and apps in GitHub Marketplace to enhance your workflow.
  • Participating in Hackathons
    Join GitHub-hosted hackathons to collaborate and showcase coding skills.
  • GitHub Sponsors and Funding
    Support open-source developers through GitHub Sponsors funding program.
